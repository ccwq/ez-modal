# EZModal

`EZModal` 是一个基于 Vue 3 + TSX 的通用弹窗组件，支持组件直用与 `useEZModal` 程序化调用两种方式。

## 特性

- 支持 `v-model` 显隐控制
- 支持 `title / default / footer` 插槽
- 支持 `beforeClose` 关闭拦截
- 支持 `ESC`、遮罩点击关闭
- 支持 `appendTo` 与 `destroyOnClose`
- 支持头部拖动，并兼容缩放容器位移补偿

## 快速开始

```bash
npm i ez-modal vue
```
